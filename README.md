## 📖 Overview

The **Snake and Ladder Game** is a classic board game where two players compete to reach square **100**. Players roll a six-sided die, climb ladders to advance faster, and avoid snakes that move them backward. This project is implemented as a **console application** in C and is ideal for beginners learning programming and game development concepts.

---

## ✨ Features

* 🎮 Two-player gameplay
* 🎲 Random dice rolling using `rand()`
* 🐍 Snakes that move players down
* 🪜 Ladders that help players climb up
* 📍 Dynamic player movement
* 🖥️ Console-based game board display
* 🏆 Automatic winner announcement
* ⚡ Beginner-friendly and well-structured code

## 🎮 Game Rules

1. Both players start at **position 0**.
2. Players take turns rolling a six-sided die.
3. Landing on a **ladder** moves the player upward.
4. Landing on a **snake** moves the player downward.
5. Players cannot move beyond square **100**.
6. The first player to reach **100** wins the game.
